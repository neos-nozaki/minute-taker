# テスト用音声ファイル置き場

このディレクトリにテスト用の音声ファイルを配置してください。

## 推奨ファイル

### 1. 超短時間ファイル（10-30秒）
開発・動作確認用。APIコストが最小で高速イテレーションが可能。

**ファイル名例**: `test_short.wav`

### 2. 中程度ファイル（3-5分）
機能テスト、プロンプト最適化用。

**ファイル名例**: `test_medium.wav`

### 3. 長時間ファイル（30-60分）
本番想定の負荷テスト用。（開発初期は不要）

**ファイル名例**: `test_long.wav`

## 対応フォーマット

- `.wav` （推奨：無圧縮、高品質）
- `.mp3` （推奨：圧縮、ファイルサイズ小）
- `.m4a`, `.flac`, `.ogg`, `.webm` 等も対応

## 注意事項

⚠️ `.gitignore` で音声ファイルはGit管理から除外されています。
⚠️ 機密情報を含むファイルは使用しないでください（OpenAI APIに送信されます）。
